# Support

Guillotine is an early OSS project. Use the repository issue tracker for:

- reproducible bugs,
- generated DSL quality issues,
- OpenAPI specs that fail to compile,
- confusing generated errors or docs,
- release packaging problems.

Please include:

- Guillotine version or commit,
- Python version,
- the OpenAPI spec or a minimal redacted reproduction,
- the exact `guillotine build` or `guillotine inspect` command,
- generated traceback or incorrect generated DSL snippet.

Security issues should follow [SECURITY.md](SECURITY.md), not public issues.
