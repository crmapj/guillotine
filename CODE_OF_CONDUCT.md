# Code of Conduct

Guillotine uses the [Contributor Covenant](https://www.contributor-covenant.org/)
as the baseline for project participation.

## Expected Behavior

- Be respectful and direct.
- Assume contributors are trying to improve the project.
- Keep technical disagreement focused on evidence, trade-offs, and user impact.
- Avoid harassment, personal attacks, discriminatory language, or disruptive conduct.

## Enforcement

Maintainers may remove comments, issues, pull requests, or other contributions that
violate this code of conduct. Repeated or severe violations may result in a temporary
or permanent ban from project spaces.

Report conduct concerns to the repository maintainers through a private contact channel
rather than a public issue.
